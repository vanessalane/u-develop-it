# u-develop-it

## Description
A Node/Express/MySQL app to manage an election!

## Usage
1. Clone the repo
2. Run `npm -i` to install dependencies
3. Run `npm start` to run the app

## Testing
After cloning the repo, run `npm test` to execute Jest tests
